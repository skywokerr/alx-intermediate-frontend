Additional Information

This responsive layout uses Tailwind's flexbox utilities. Resize your browser to see how the layout changes on mobile devices.

Implementation Details

Main Container

class="flex"

Uses flexbox to create a horizontal layout by default.

Sidebar

class="w-1/3 bg-gray-300 p-4"
  • w-1/3: Sets width to 1/3 of the container
  • bg-gray-300: Sets background color
  • p-4: Adds padding

Content Section

class="w-2/3 bg-gray-500 p-4"
  • w-2/3: Sets width to 2/3 of the container
  • bg-gray-500: Sets background color
  • p-4: Adds padding

Responsive Design

@media (max-width: 768px) {
  main {
    flex-direction: column;
  }
}

On screens smaller than 768px, the layout changes to vertical stacking.
