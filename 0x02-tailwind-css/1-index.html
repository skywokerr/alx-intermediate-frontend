How This Was Created

This responsive grid layout was created using Tailwind CSS classes:

class="grid grid-cols-3 gap-4" class="bg-blue-200 p-4" class="bg-blue-300 p-4" class="bg-blue-400 p-4"

The layout is responsive and will switch to a single column on screens smaller than 768px:

@media (max-width: 768px) { main { grid-template-columns: 1fr; } }
