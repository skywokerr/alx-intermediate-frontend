Implementation Details

Navigation Container

class="flex space-x-5 justify-center bg-gray-300 text-white p-4"
  • flex: Enables flexbox
  • space-x-5: Adds horizontal spacing between children
  • justify-center: Centers the navigation items
  • bg-gray-300: Sets the background color
  • text-white: Sets the text color
  • p-4: Adds padding

Navigation Items

class="px-2 text-xl hover:bg-gray-500 p-2 rounded-lg"
  • px-2: Adds horizontal padding
  • text-xl: Sets the font size
  • hover:bg-gray-500: Changes background on hover
  • p-2: Adds padding
  • rounded-lg: Rounds the corners

Responsive Design

@media (max-width: 768px) {
  nav {
    flex-direction: column;
    align-items: flex-start;
  }
}

On screens smaller than 768px, the navigation changes to a vertical layout with items aligned to the start.
