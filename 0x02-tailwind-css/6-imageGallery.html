Implementation Details

Main Container

class="flex"

Uses flexbox to create a horizontal layout by default.

Image Gallery Grid

display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 1rem;

Creates a 3-column grid layout with gaps between items.

Responsive Design

@media (max-width: 768px) {
  .gallery {
    grid-template-columns: 1fr;
  }
}

On screens smaller than 768px, the gallery changes to a single column layout.

Responsive Images

.gallery-img {
  width: 100%;
  height: 200px;
  object-fit: cover;
}

Images will resize properly to fit their containers while maintaining aspect ratio.
